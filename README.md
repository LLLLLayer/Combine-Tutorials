## 第一部分：Combine 简介

在本书的这一部分中，你将了解 Combine 的基础知识并了解它所包含的一些构建块。 你将了解 Combine 旨在解决什么问题，以及它提供了哪些抽象来帮助你解决这些问题：Publisher、Subscriber、Subscription、Subject 等等。
